# Sounds Directory

This directory contains audio assets for the "Through the Window" interactive experience.

## Required Assets

- **ambient.mp3** - Main ambient background audio track

## Additional Sound Effects (Optional)

- Suspenseful stingers
- Footstep sounds
- Door creaks
- Whispers or breathing sounds
- Environmental ambience

## Audio Guidelines

- Format: MP3 (for compatibility) or OGG
- Keep file sizes reasonable (compressed but quality)
- Loop ambient tracks seamlessly
- Normalize audio levels across all files
- Horror/suspense theme with subtle, unsettling tones

## Adding Audio

Place your audio files directly in this directory. Ensure the main ambient track is named `ambient.mp3`.
