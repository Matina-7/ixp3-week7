# Videos Directory

This directory contains video assets for the "Through the Window" interactive experience.

## Required Assets

- **feed1.mp4** - Security camera feed 1 (left side view)
- **feed2.mp4** - Security camera feed 2 (right side view)

## Video Guidelines

- Format: MP4 (H.264 codec for broad compatibility)
- Resolution: 1280x720 or 1920x1080
- Keep file sizes optimized (compressed but quality)
- Videos should loop seamlessly
- Security camera aesthetic (grainy, monochrome, timestamp overlay optional)
- Duration: 30-60 seconds loop

## Video Content Suggestions

- Empty rooms or corridors
- Subtle movements or shadows in the background
- Occasional figure appearances
- Static/glitch effects for horror atmosphere
- Low contrast, surveillance camera look

## Adding Videos

Place your video files in this directory. Name them according to the convention:
- `feed1.mp4`, `feed2.mp4`, etc.
