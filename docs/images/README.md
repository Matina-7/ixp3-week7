# Images Directory

This directory contains image assets for the "Through the Window" interactive experience.

## Required Assets

- **loading.gif** - Loading animation displayed during initialization
- **icons/** - UI icons and interface elements
- **placeholders/** - Placeholder images for development

## Image Guidelines

- Use optimized formats (WebP, PNG, JPG)
- Keep file sizes minimal for faster loading
- Use dark themes consistent with horror aesthetic
- Recommended color palette: blacks, dark reds (#8b0000), grays

## Adding Images

Place your image files directly in this directory or organize them in subdirectories as needed.
